Installing open jdk

1. sudo apt update
2. sudo apt install default-jdk
3. java -version

Downloading apache maven

1. wget https://www-us.apache.org/dist/maven/maven-3/3.6.3/binaries/apache-maven-3.6.3-bin.tar.gz -P /tmp
2. sudo tar xf /tmp/apache-maven-*.tar.gz -C /opt

setting up the environment variables

1. sudo nano /etc/profile.d/maven.sh

Paste the following code in it
export JAVA_HOME=/usr/lib/jvm/default-java
export M2_HOME=/opt/maven
export MAVEN_HOME=/opt/maven
export PATH=${M2_HOME}/bin:${PATH}
save and close

2. sudo chmod +x /etc/profile.d/maven.sh
3. source /etc/profile.d/maven.sh

verify the installation

1. mvn -version
